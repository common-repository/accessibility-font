== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `accessibilityfont.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates

== Frequently Asked Questions ==

= A question that someone might have =

- ¿Cómo puedo utilizar este plugin? -

Siga las instrucciones de instalación. A continuación se presentan las opciones que puede establecer en el widget: -

* Se establece en estos tamaños: *
Aquí es donde se establece el tamaño de fuente que desea habilitar para los usuarios a utilizar (leer sobre la propiedad font-size CSS en W3Schools). El grupo font-size más simple que puede utilizar son más pequeños, heredar, más grande.

* Texto controlador Set: *
De forma predeterminada, el widget muestra la letra "T" en función del número de tamaños de fuente se utiliza; es decir, si se establece sólo dos font-size, entonces sólo dos letras aparece T, y así sucesivamente. 
- ¿Puedo utilizar esto en una página web sin WP? -

Si se puede, hay que descargar nuestro plugin específico para webs sin WP en http://www.aquicoco.com/accesibilityfont.
